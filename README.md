# DalamudPlugins

个人用卫月插件仓库

## 插件列表

* [Allagan Tools](https://github.com/HirotaZX/InventoryTools) 亚拉戈工具 fork，提供物品列表等功能，Artisan 雇员功能的依赖，进行了国服适配

## 使用

1. 打开卫月设置-测试版
2. 复制粘贴下面的仓库链接
3. 点击 + 号按钮
4. 点击保存图标

```
https://raw.githubusercontent.com/HirotaZX/DalamudPlugins/main/repo.json
```
